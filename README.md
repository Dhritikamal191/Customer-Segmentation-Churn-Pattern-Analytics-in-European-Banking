# Customer-Segmentation-Churn-Pattern-Analytics-in-European-Banking

# Overview
This project focuses on analyzing customer churn behavior in European Bank Dataset using Exploratory Data Analysis (EDA) and data visualization techniques. The objective is to identify key factors influening customer attrition and provide actionable insights for improving cusomer retention.

# Objectives
- Analyze customer churn patterns
- Identify key facors affecting churn
- Perform distribtuion and contribution analysis
- Build an interactive dashboard for insights.

# Dataset Description
The dataset contains customer information including:
- Demographic details (Age, Gender, Geography)
- Financial attributes (Balance, Estimated Salary)
- Behavioral data (Tenure, Number of Products, Activity Status)
- Target Variable: Exited (Churn)
  
# Tools & Technologies Used
- Pthon
- Pandas
- Numpy
- Matplotlib / Plotly
- Streamlit
  
# Exploratory Data Analysis (EDA)
- Univariate and bivariate analysis
- Distribution analysis of churn across segments.
- Contibution analysis to identify high-impact groups.
- Visualization using charts and graphs.

# Key Insights
- Behavioral factors like customer activity and product usage significantly influence churn.
- New customers show slightly higher churn tendency
- Salary and tenure have limited impact on churn.
- Inactive customers are more likely to exit.

# Dashboard
An interactive dashboard was developed using Streamlit to:
- Display key performance indicators (KPIs)
- Visualize churn patterns across segments
- Enable dynamic data exploration.
